Задание заключается в том чтобы создать ASP.NET MVC приложение:
Предметная область: запись на прием к врачу. 
Пользователи: регистратура и люди которые хотят записаться на прием (Обычный пользователь). 
Пользователь регистратуры должен иметь возможность добавить врача и проставить его график работы (шаг по 20 минут)
Пользователь регистратуры должен иметь возможность добавить отделение и к нему прикрепить врача 
Пользователь регистратуры должен иметь возможность завести карточку гражданина и иметь возможность добавить туда болезни
Обычный пользователь должен иметь возможность просмотреть список врачей и записаться на прием. При этом есть фильтрация по отделению, ФИО врача
Авторизацию и регистрацию можно использовать “из коробки”
 
Стек технологий для Backend-разработчика:
Code first with Entity Framework 6 / Entity Framework Core 
ASP.NET MVC 5 / ASP.NET MVC Core
Razor для работы с версткой
Плюсом будет использование технологии AJAX (AngularJS, Angular 4)
